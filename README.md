# skkeletal.vim

## Introduction

**skkeletal.vim** は、日本語入力システム [skkeleton](https://github.com/vim-skk/skkeleton)に関連する、個人的に `.vimrc(init.vim)` に書きたくない設定をプラグインにしたものです。

現在同梱している機能は以下のとおりです。

    - 辞書のダウンロード機能
    - Dvorak配列ユーザー向けの設定
